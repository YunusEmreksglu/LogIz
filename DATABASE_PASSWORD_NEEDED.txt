⚠️ ÖNEMLİ: Database Şifresi Gerekli!

Şimdi Supabase Database şifrenizi almanız gerekiyor:

1. Supabase Dashboard'a gidin: https://supabase.com/dashboard
2. Sol menüden "Project Settings" (⚙️) tıklayın
3. "Database" sekmesine gidin
4. "Connection string" bölümünde:
   - "URI" formatını seçin
   - Connection string'i göreceksiniz
   - URL'de [YOUR-PASSWORD] kısmını göreceksiniz

5. Eğer şifrenizi kaybettiyseniz:
   - "Reset database password" butonuna tıklayın
   - Yeni şifre oluşturun
   - Güvenli bir yere kaydedin

6. .env.local dosyasında [YOUR-PASSWORD] yazan yerleri şifrenizle değiştirin:
   - DATABASE_URL içindeki [YOUR-PASSWORD]
   - DIRECT_URL içindeki [YOUR-PASSWORD]

Örnek:
DATABASE_URL="postgresql://postgres.tmavagzxznmmwecbudux:MySecretP@ssw0rd@aws-0-eu-central-1.pooler.supabase.com:6543/postgres?pgbouncer=true&connection_limit=1"

⚠️ Eğer şifrede özel karakterler varsa URL encode edin:
@ → %40
# → %23
& → %26

Sonra şu komutu çalıştırın:
npx prisma db push
